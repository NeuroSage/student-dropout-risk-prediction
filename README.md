# Student Dropout Risk Prediction

This repository contains the capstone project for the Postgraduate Diploma in AI and ML.

The project focuses on predicting student dropout risk using supervised machine learning, with an emphasis on decision support, return on investment (ROI), risk management, and governance for educational institutions.

## Repository Structure
- notebooks/ : Jupyter notebooks for analysis and modeling
- data/ : Datasets and data references
- models/ : Saved model artifacts
- src/ : Supporting scripts (if applicable)
- reports/ : Final report and presentation decks

## Reproducibility
All experiments are designed to be reproducible using the provided notebooks and documented dependencies.
